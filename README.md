# PedeFlash

PedeFlash é um app mobile de delivery de comida desenvolvido com React Native, Expo e TypeScript.

O projeto está em fase inicial e tem como objetivo praticar a construção de interfaces mobile, navegação entre telas e renderização de listas de produtos vindos futuramente de um backend.

A proposta é criar uma experiência simples e moderna para explorar restaurantes, visualizar produtos e simular o fluxo de pedido em um aplicativo de comida.

## Status do projeto

Em desenvolvimento.

Funcionalidades atuais:

- Navegação inferior com abas
- Tela inicial
- Cards de produtos
- Lista de produtos com FlatList
- Estrutura inicial para novas páginas
